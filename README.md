TP-Redis
========

TP - SIR - Redis

Quelles sont les types de données stockés dans Redis, que peut on faire comme types de requêtes ?

Voici les différents types de données stockés dans Redis et les requêtes possibles associées à ces types :

String
- SET
- INCR
- GETSET
- GET

Listes
- RPUSH
- LRANGE

Hash
- HSET
- HGET
- HGETALL
- HKEYS
- HVALS
- HLEN

Sets
- SADD
- SCARD
- SINTER
- SUNION
- SDIFF

Sets triés
- ZADD
- ZRANGE
- ZREVRANGE
- ZRANGEBYSCORE
